## ✨ Features

<!-- Core real-time audio processing -->
- **Real-time processing** — low-latency voice transformation powered by PortAudio
<!-- List of available voice effects -->
- **Built-in effects:**
  - 🎵 Pitch Shift — raise or lower voice pitch
  - 🔊 Echo / Reverb — add room or hall reverb
  - 🤖 Robot Voice — metallic robotic sound
  - 🗣️ Deep Voice — lower pitch with resonance
  - 🐿️ Chipmunk — high-pitch squeaky voice
  - 🎛️ Custom EQ — manual frequency band control
<!-- GUI description -->
- **Simple GUI** — lightweight Tkinter interface, no browser required
<!-- Real-time parameter adjustment -->
- **Live controls** — adjust effect parameters with sliders in real time
<!-- OS support -->
- **Cross-platform** — runs on Windows, macOS, and Linux
